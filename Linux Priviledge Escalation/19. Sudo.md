One of the latest vulnerabilities for sudo carries the `CVE-2021-3156` and is based on a heap-based buffer overflow vulnerability. This affected the sudo versions:

 - 1.8.31 - Ubuntu 20.04
 - 1.8.27 - Debian 10
 - 1.9.2 - Fedora 33
 - and others

```bash
$ sudo -V | head -n1

Sudo version 1.8.31
```
